# Setting up
- Install composer
- Install nodeJs
- Install PHP
- Open CMD and go to root directory of cloned project
- Type `composer install`  and press enter
- Type `npm install` and press enter
- Type `npm run serve` and press enter
- Wait for browser to load your project
